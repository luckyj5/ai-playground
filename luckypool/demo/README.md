# LuckyPool — Demo Recording

`luckypool-demo.mp4` is a ~37-second walkthrough of LuckyPool covering:

1. Home — hero with subtle Inter typography, real 8-ball logo, three discipline chips
2. Disciplines — felt tables with **horizontally-oriented** racks: 8-ball triangle (apex 1, 8 in centre), 9-ball diamond (apex 1, 9 in centre), snooker break (reds with apex pointing at the pink)
3. 9-Ball Quickfire CTA banner
4. Global pulse — stat strip + top-3 per discipline
5. `/play` — Quickfire run (click 1 → 9 in order, completion overlay, sidebar `You` rank insertion)
6. `/leaderboard` — Overall tab (Ronnie O'Sullivan top 898) → 9-Ball tab → Quickfire tab with `You · <time>` highlighted in gold

Captured at 1600x1080 / 25fps. Re-run from this branch with `npm run dev` to see it live.
