# LuckyPool — Demo Recording

`luckypool-demo.mp4` is a ~47-second walkthrough of LuckyPool covering:

1. Home — hero, real 8-ball logo, three discipline chips
2. Disciplines — felt tables with 8-ball triangle, 9-ball diamond, snooker break racks
3. 9-Ball Quickfire CTA banner
4. Global pulse — stat strip + top-3 per discipline
5. `/play` — full Quickfire run (click 1 → 9, completion overlay, sidebar `You` rank insertion)
6. `/leaderboard` — Overall tab (Ronnie O'Sullivan top 898) → 9-Ball tab → Quickfire tab with `You · <time>` highlighted

Captured at 1600x1120 / 25fps. Re-run from this branch with `npm run dev` to see it live.
